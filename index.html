募集要項

はじめまして。私は大阪の弱小ベンチャーでIT関連の業務を担当をしている者です。
（社名等明示できず申し訳ありません。面談にお越しいただいた際にはきちんと自己紹介します）
今回のお誘いについて、会社の状況も含めて率直なところを以下に記載します。

# 会社の状況と直近のニーズ

大阪の小さな会社です。IT関連事業としては、お客から仕事をもらって何かしら作る受託開発と呼ばれる事業と自社サービス開発運営事業の2つがあります。 これは現在社長と私とで対応していますが、最近受託案件を増やしており手が足りなくなってきています。ここを助けてくれる方を探しています。

経験は無くとも構いません。あなたにIT系の仕事がしたいという意思があればOKです。
ちなみに私が想定している「あなた」は、ニートであるとか先の見えない仕事に就いてるだとかで何らか将来に危機感を抱いている人です。 「なんとかして手に職つけなきゃヤバイ」とかそういう危機感みたいな強い意思は持っていてほしいです。

# ご提供できる業務機会

「Web制作」と呼ばれる工程に係る作業のすべてを経験していただけます。小さな会社で働くことのメリットの一つがこれです！ここではあなたの役割は限定されません。いや、Web系という括りでは限定されますが、エンジニアとしての基礎的な力を広く浅く蓄えることができます。

ただ最初からいろいろやらされても死ぬだけなので、まずはLP（ランディングページ）コーディングといって、それほど難易度の高くないHTMLページの作成から入っていただくと思います。 これはある程度既存の見本を参考にして進められます。ここでHTML／CSS／JavaScriptという、Web制作の基本的な要素を学んでいただけます。

その他、いわゆるサーバーサイドの開発（PHPとかJavaとか、あるいはサーバー環境を構築するとか）もできます。そうすると必然的にデータベースと呼ばれるものにも触ります。 デザインの仕事も取ろうと思えば取れますのでやりたければどうぞ（私はできません）。

# 勤務地

勤務地は大阪市内です。タワーマンションの一室で私服勤務が可能です。

# （想定）労働条件

もしあなたが未経験者の場合はアルバイトとしてスタートできないか交渉させていただくことになると思います。時給は1,000円を考えています（調整できます）。 これは正直に言って会社側のリスクヘッジを考えてのことです。やはりどうしても「長く続けてもらえるか？」という点について懐疑的にならざるを得ないと思うのです。 例え面談をしっかりしたとしてもです。

なので、「まずは自分が楽しくできそうか試してみませんか？」
…少々欺瞞的ですか？Web制作の楽しさを感じる機会は間違いなく提供できます。こうした楽しみとうまくできない時の苦しみを天秤にかけて自分がやっていけそうかどうか実感していただきたいのです。

勤務時間は平日の9:00〜18:00です。受託開発には納期が存在しますので、場合によっては残業が発生して日に10時間以上働いてもらうこともあり得ます。

つまり条件は良くありません。楽に稼げる割の良い仕事ではないと思います。やりがい搾取的な物言いになってしまいますが、IT業界にエントリーするための一種の修行と考えていただく（自分に言い聞かせていただく）必要はあると思います。

ただ、お金とか労働時間以外であなたにお渡しできる価値はあります。それは最後にまとめます。

# 福利厚生、なし！w

福利厚生的なものは特段ありません。 無理やり挙げるとすれば、たまに社長が昼食を奢ってくれるのとミネラルウォーターが常備されているので飲み物を買う必要がないことくらいでしょうか。 あと歓迎会等でたまに飲み会がありますがそれは会社のお金が使われます。

# その他

少人数の会社ですがリア充が多いです。もしあなたがなんJ彼女欲しい部などのスレに頻繁に書き込みしているような人だとすると最初は馴染みにくさを感じるかもしれません。
しかしそれは私も似たようなものなのでたぶん大きな問題にはならないと思います。すぐに慣れます。
社長はリア充の権化みたいな人ですが同時によく出来た人なのであなたにとっても魅力的に映るはずです。

# 会社にとって何が良いのか

失礼な言い方になってしまうかもしれませんが、低リスクで掘り出しものが見つかるかもしれないことです。 本当に率直に言って、知り合いでもない人をそれもなんJきっかけで雇うとかあまり正気の沙汰ではないと思ってます。

しかしなんJにたまにいる切羽詰まってる人がその危機感をやる気に変換できたらこの業界ですごく成長するのではないかと思うのです。 そうした人が来てくれれば、会社にとっては案件が多くこなせて利益に繋がって良いですね。

# あなたにとって何が良いのか

  • 業界にチャレンジできる（合う合わない確かめられる）
  • 小規模な会社の割に身のある仕事ができる（常駐派遣とかテスターではない）
  • 広く浅く基礎力をつけられる

つまりステップアップにつながる経験をしていただけます。この小さな会社は1年後どうなっているかわかりませんが、あなた個人にスキルがあれば業界を渡っていけます。この会社で働いていただくことでスキルをつける手助けができると思っています。 会社の利益的には長くいて欲しいのですが、スキルをつける過程で活躍してもらえればまぁOKです。

半分余談ですがさらにプラスの経験もできます（興味があれば）。業務設計とかマーケティングとかには興味ありますか？上で挙げたような経験・スキルはつぶしが効いて食いっぱぐれないとは思うのですが、そこまでレアな価値ではないです。
今やっている受託開発も価格競争に晒されて割りに合わなくなっていきます。なのでエンジニアとしてプラスアルファの売りが必要になってきます。

この会社は自社サービスを運営しています。そこにエンジニアとして関わると、受託開発ではできない経験ができます。そもそもの業務設計だったりマーケティングの仕掛け作りだったりです。
こういう経験を持っていると、受託ではない仕事・会社への道が拓けます。自社で商売しているような会社の社内SE的なポジションですね。このポジションもいいのかわるいのか分かりませんが、そんじょそこらの会社で働くよりもより広い選択肢を持てるようになるかもしれません。こんなところです。
